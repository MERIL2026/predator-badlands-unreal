# Predator Badlands (Unreal Engine Project)

## 🎮 Overview
Predator Badlands is an open-world environment project built using Unreal Engine.  
It features realistic terrain, lighting, and cinematic visuals.

## 🚀 Features
- Realistic terrain generation  
- Dynamic lighting & shadows  
- Environmental assets (rocks, grass, vegetation)  
- Third-person character setup  
- Cinematic environment design  

## 🛠️ Tech Used
- Unreal Engine 5  
- Blueprint System  
- Environment Design  
- Lighting & Rendering
